# Resources for A Comprehensive Review of RGB/RGB-D-Based 6D Object Pose Estimation Techniques

This repository is a companion resource for the manuscript entitled **“A Comprehensive Review of RGB/RGB-D-Based 6D Object Pose Estimation Techniques”**, submitted to *The Visual Computer*.

## Contents
- `papers.md`: representative papers related to 6D object pose estimation
- `datasets.md`: public datasets commonly used in RGB/RGB-D-based 6D pose estimation
- `metrics.md`: representative evaluation metrics used in the literature

## Purpose
This repository is intended to improve the transparency, traceability, and usability of the reviewed literature, datasets, and evaluation metrics summarized in this survey.

This repository is directly related to the above manuscript. If you find these resources useful, please cite the corresponding paper.
